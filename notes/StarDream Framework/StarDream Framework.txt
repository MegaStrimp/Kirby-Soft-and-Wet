StarDream Framework

"Forever's Gonna Start Tonight"
StarDream Framework is a Game Maker package made by Strimp's Kitchen to ease development for indie games and fan projects.

Adding A New Attack

1 - Add the attack to 'enum playerAttacks' in the Player's Create Event.
2 - Make a script for executing the attack. (scr_MKSS_ExecuteAttack_Player_)
3 - Add the execution script to the Player's base code.
